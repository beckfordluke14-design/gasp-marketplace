## 2. THE ASSET SOVEREIGNTY (STORAGE)
- **Local Vault**: `C:\gasp_assets\personas\{persona_id}`
- **Tiered Optimization**:
  - **Feed Images**: WebP (Q85) for global speed.
  - **Vault Images**: High-Fidelity JPG (Q95) for subscriber quality.

## 3. THE EDGE ROUTING (NETWORKING)
- **Direct Edge**: `https://asset.gasp.fun/personas/{pid}/{file}`
- **Proxy Node**: Cloudflare edges cache these VPS assets.
- **DB Fallback**: If `asset.gasp.fun` is down, failback to Supabase Storage.

## 4. THE BIRTH PROTOCOL (DATABASE)
- **Table: personas**: Requires `id`, `name`, `agency_id`, `system_prompt`, and `is_active`.
- **Table: posts**: Requires `persona_id`, `content_url`, `is_vault`, and `is_burner` (Featured).
- **Status**: Always set `is_active: true` for the identity to appear on the frontend.
